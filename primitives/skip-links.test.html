Accessibility Features

  • ✅ Visually hidden until focused (no layout shift)
  • ✅ High contrast focus indicators
  • ✅ Smooth scroll to target (respects prefers-reduced-motion)
  • ✅ Focus management for screen readers
  • ✅ ARIA navigation role
  • ✅ Keyboard accessible (Tab + Enter)
  • ✅ Customizable via CSS custom properties
  • ✅ Zero JavaScript required for basic function

Performance Characteristics

  • Zero render cost when not focused (CSS-only)
  • No JavaScript execution until link clicked
  • Shadow DOM encapsulation prevents style conflicts
  • Minimal DOM footprint
