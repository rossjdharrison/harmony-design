Font Size Scale

The Harmony Design System uses an 8-step font size scale optimized for UI design. Each size includes carefully calculated line heights for optimal readability. The scale ranges from 10px (xs) to 30px (3xl), providing flexibility for all typography needs.

Usage Example

// Import typography tokens
import { getFontSize, getTextStyle } from './tokens/typography.js';

// Get a single font size
const headingSize = getFontSize('xl'); // '20px'

// Get complete text style
const headingStyle = getTextStyle('xl', 'bold');
// { fontSize: '20px', lineHeight: '28px', fontWeight: '700', letterSpacing: '0' }

// Apply to element
element.style.fontSize = getFontSize('base');

// Use in CSS custom properties
document.documentElement.style.setProperty('--font-size-heading', getFontSize('2xl'));
