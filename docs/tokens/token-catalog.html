Token Catalog

Visual reference for all design tokens in the Harmony Design System

Primary Color Scale

The primary color scale provides 11 shades for the main brand color. Use lighter shades (50-200) for backgrounds and surfaces, middle shades (400-600) for interactive elements and CTAs, and darker shades (700-950) for text and emphasis.

Common Usage Patterns

  • Primary buttons: 500-600
  • Button hover states: 400-500
  • Button active/pressed: 600-700
  • Background tints: 50-100
  • Links and interactive text: 600-700

Neutral/Gray Scale

The neutral scale provides 11 achromatic shades for surfaces, text, and UI elements. This scale is essential for establishing visual hierarchy and ensuring proper contrast ratios. Lighter shades (50-200) work for backgrounds, middle shades (300-600) for secondary elements and borders, and darker shades (700-950) for text.

Common Usage Patterns

  • Page backgrounds: 50-100
  • Card surfaces: 100
  • Borders and dividers: 200-300
  • Disabled states: 300-400
  • Placeholder text: 400
  • Secondary text: 500-600
  • Body text: 600-700
  • Headings: 700-800
  • Primary text: 900-950
