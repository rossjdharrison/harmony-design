Spacing Scale

The Harmony Design System uses a consistent spacing scale based on a 4px base unit. This creates visual rhythm and maintains consistency across all components and layouts. The scale progresses from micro spacing (4px) to large layout gaps (96px).

Base Scale

Scale Guidelines

  • 0: No spacing - use for resets and zero-gap layouts
  • 1-3 (4-12px): Micro spacing - inside components, between text and icons
  • 4-6 (16-24px): Small spacing - component padding, inline gaps
  • 8-12 (32-48px): Medium spacing - between components, section padding
  • 16-24 (64-96px): Large spacing - page sections, major layout gaps

Semantic Spacing Tokens

Use semantic tokens for consistent component spacing. These map to scale values but provide meaningful names.

Usage Examples

JavaScript

// Import spacing functions import { getSpacing, getSemanticSpacing } from './tokens/spacing.js'; // Use scale values const gap = getSpacing(4); // 16px // Use semantic tokens const padding = getSemanticSpacing('componentPaddingMedium'); // 16px // Apply to element element.style.padding = `${padding}px`;

CSS Custom Properties

/* Use CSS variables */ .component { padding: var(--spacing-4); /* 16px */ gap: var(--spacing-gapMedium); /* 16px */ } .section { margin-bottom: var(--spacing-12); /* 48px */ }
