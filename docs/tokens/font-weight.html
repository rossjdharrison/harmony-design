Font Weight Tokens

Typography weight scale for the Harmony Design System

Purpose: Font weight tokens provide consistent typographic hierarchy and emphasis across the design system.

Scale: Four carefully chosen weights from regular (400) to bold (700) that work well across different font families and display contexts.

Implementation: CSS custom properties defined in styles/tokens-font-weight.css

Usage Examples

CSS Custom Properties

.component-title { font-weight: var(--font-weight-semibold); } .body-text { font-weight: var(--font-weight-regular); } .emphasis { font-weight: var(--font-weight-medium); }

Utility Classes

<h1 class="font-weight-bold">Page Title</h1> <h2 class="font-weight-semibold">Section Heading</h2> <p class="font-weight-regular">Body text</p> <span class="font-weight-medium">Emphasized text</span>

Web Components

class HarmonyButton extends HTMLElement { connectedCallback() { this.shadowRoot.querySelector('button').style.fontWeight = 'var(--font-weight-medium)'; } }

Design Guidelines

When to Use Each Weight

  • Regular (400): Default for all body text, form inputs, and standard UI elements
  • Medium (500): Subtle emphasis within paragraphs, active states, secondary headings
  • Semibold (600): Primary headings, important buttons, key information
  • Bold (700): Page titles, critical alerts, maximum emphasis (use sparingly)

Best Practices

  • Maintain clear hierarchy: Use only 2-3 weights per page to avoid visual confusion
  • Consider readability: Heavier weights work better at larger sizes
  • Test across fonts: Not all typefaces support all weights equally well
  • Avoid extremes: 400-700 range provides sufficient contrast without compromising legibility
  • Combine with size: Weight and size work together to create hierarchy

Accessibility

  • Font weight alone should not convey critical information (combine with color, size, or labels)
  • Ensure sufficient contrast between weights for clear visual hierarchy
  • Test readability at different zoom levels and screen resolutions
  • Consider dyslexic users: Avoid very light or very heavy weights for body text
