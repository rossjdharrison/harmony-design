Shadow Tokens

Elevation levels for visual hierarchy and depth

Token Scale

Shadow tokens create visual hierarchy through elevation. Higher elevation levels indicate more importance or interactivity.

Usage Guidelines

  • none: Flat elements, on-surface components, disabled states
  • sm: Subtle elevation for cards at rest, list items
  • default: Standard elevation for buttons, cards, interactive elements
  • md: Raised elements like dropdowns, popovers, tooltips
  • lg: Modal dialogs, drawers, prominent overlays
  • xl: Critical notifications, alerts, top-level modals

Code Usage

JavaScript Import

import { SHADOW_TOKENS, applyShadowTokens } from './tokens/shadow.js'; // Apply to element element.style.boxShadow = SHADOW_TOKENS.default; // Apply as CSS custom properties applyShadowTokens(document.documentElement);

CSS Custom Properties

.card { box-shadow: var(--shadow-default); } .modal { box-shadow: var(--shadow-lg); } .button:hover { box-shadow: var(--shadow-md); }
