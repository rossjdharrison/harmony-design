Easing Tokens

Animation timing functions for consistent motion design

Standard Easing Functions

Linear

Constant speed, no acceleration

linear

transition: all 300ms var(--easing-linear);

Ease In

Slow start, accelerating

cubic-bezier(0.4, 0, 1, 1)

transition: all 300ms var(--easing-ease-in);

Ease Out

Fast start, decelerating

cubic-bezier(0, 0, 0.2, 1)

transition: all 300ms var(--easing-ease-out);

Ease In Out

Slow start and end, fast middle

cubic-bezier(0.4, 0, 0.2, 1)

transition: all 300ms var(--easing-ease-in-out);

Spring Presets

Spring Gentle

Subtle spring effect, low bounce

cubic-bezier(0.25, 0.1, 0.25, 1)

transition: all 400ms var(--easing-spring-gentle);

Spring Default

Standard spring, medium bounce

cubic-bezier(0.34, 1.56, 0.64, 1)

transition: all 400ms var(--easing-spring-default);

Spring Bouncy

Pronounced spring, high bounce

cubic-bezier(0.68, -0.55, 0.265, 1.55)

transition: all 500ms var(--easing-spring-bouncy);

Spring Snappy

Quick, tight spring, minimal bounce

cubic-bezier(0.5, 1.25, 0.75, 1.25)

transition: all 350ms var(--easing-spring-snappy);
