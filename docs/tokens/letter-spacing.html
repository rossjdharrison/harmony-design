Letter Spacing Tokens

Letter spacing (tracking) values for typography control. These tokens affect the horizontal space between characters, impacting readability and visual density.

Token Reference

Tight -0.02em
Display Heading

Tighter letter spacing is typically used for large headings and display text where the increased font size naturally creates more space between letters. Reducing the spacing slightly improves visual cohesion.

Best for: Headings (h1-h3), display text, titles, large UI labels
Normal 0
Standard Heading

Normal letter spacing uses the default browser spacing, which is optimized for most body text and general UI elements. This is the most common setting and provides good readability for standard font sizes.

Best for: Body text, paragraphs, default UI components, buttons
Wide 0.02em
Spacious Heading

Wider letter spacing improves readability for small text, uppercase text, and situations where enhanced legibility is needed. The increased spacing creates a more open, airy feel.

Best for: Small text, uppercase labels, captions, metadata, accessibility

Usage Examples

JavaScript

import { LETTER_SPACING } from './tokens/letter-spacing.js';

// Apply to element
element.style.letterSpacing = LETTER_SPACING.tight;

// Use in Web Component
class MyComponent extends HTMLElement {
  constructor() {
    super();
    this.attachShadow({ mode: 'open' });
    this.shadowRoot.innerHTML = `
      <style>
        h1 { letter-spacing: var(--letter-spacing-tight); }
        p { letter-spacing: var(--letter-spacing-normal); }
        .caption { letter-spacing: var(--letter-spacing-wide); }
      </style>
    `;
  }
}

CSS

/* Using CSS variables */
.heading {
  letter-spacing: var(--letter-spacing-tight);
}

.body-text {
  letter-spacing: var(--letter-spacing-normal);
}

.small-caps {
  text-transform: uppercase;
  font-size: 0.875rem;
  letter-spacing: var(--letter-spacing-wide);
}

Design Guidelines

  • Tight spacing works best with larger font sizes (18px+) where natural spacing is already generous
  • Normal spacing is the default and works for most use cases
  • Wide spacing improves readability for small text (12px and below) and uppercase text
  • Letter spacing is relative (em units), so it scales proportionally with font size
  • Avoid extreme letter spacing values that compromise readability
  • Consider accessibility: some users benefit from slightly wider letter spacing
