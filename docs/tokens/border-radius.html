Border Radius Scale

The border radius scale provides consistent corner rounding values for UI elements. From sharp corners to fully rounded pills, this scale covers all common use cases.

Usage Examples

import { borderRadius, getBorderRadiusCSS } from './tokens/border-radius.js';

// Direct usage
element.style.borderRadius = `${borderRadius.md}px`;

// Helper function
element.style.borderRadius = getBorderRadiusCSS('lg');

// CSS custom properties
import { injectBorderRadiusVars } from './tokens/border-radius.js';
injectBorderRadiusVars();
// Then in CSS: border-radius: var(--border-radius-md);

Design Guidelines

  • none (0px): Sharp corners for technical UI, tables, code blocks
  • sm (2px): Subtle rounding for inputs, small buttons
  • default (4px): Standard UI elements, most components
  • md (6px): Cards, panels, modal dialogs
  • lg (8px): Prominent elements, large cards
  • xl (12px): Hero sections, feature cards
  • full (9999px): Pills, badges, circular avatars
