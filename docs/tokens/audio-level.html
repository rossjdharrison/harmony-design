Level Zones

Professional Audio Standards

Audio level tokens follow professional metering standards:

  • Safe Zone (-∞ to -18dBFS): level-safe - Optimal operating range
  • Warning Zone (-18dBFS to -6dBFS): level-warning - Approaching limits
  • Peak Zone (-6dBFS to -3dBFS): level-peak - High level
  • Clip Zone (-3dBFS to 0dBFS): level-clip - Risk of distortion
  • RMS Indicator: level-rms - Average level display

Usage Example

import { 
  AUDIO_LEVEL_TOKENS, 
  getAudioLevelColor,
  getAudioLevelCSSVar 
} from './tokens/audio-level-tokens.js';

// Get color based on dBFS value
const color = getAudioLevelColor(-12); // Returns warning color

// Use CSS variables
element.style.backgroundColor = 'var(--harmony-level-safe)';

// Dynamic level metering
function updateMeter(dbfs) {
  const color = getAudioLevelColor(dbfs);
  meter.style.backgroundColor = color;
}
