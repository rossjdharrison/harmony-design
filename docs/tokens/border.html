Border Tokens

Semantic border colors for consistent UI element styling. These tokens provide appropriate contrast and visual hierarchy across light and dark themes.

Usage Guidelines

  • border-default: Use for standard borders on cards, inputs, and containers
  • border-subtle: Use for dividers and low-emphasis boundaries
  • border-strong: Use for emphasized elements or active states
  • border-focus: Use for focus indicators (keyboard navigation)

CSS Usage

/* Using border tokens with border-width tokens */ .card { border: var(--border-width-default) solid var(--border-default); border-radius: var(--radius-default); } .divider { border-bottom: var(--border-width-thin) solid var(--border-subtle); } .active-item { border: var(--border-width-thick) solid var(--border-strong); } .button:focus-visible { outline: var(--border-width-default) solid var(--border-focus); outline-offset: 2px; }

Component Example

class BorderedCard extends HTMLElement { constructor() { super(); this.attachShadow({ mode: 'open' }); } connectedCallback() { this.shadowRoot.innerHTML = ` <style> :host { display: block; border: var(--border-width-default) solid var(--border-default); border-radius: var(--radius-default); padding: 1rem; } :host(:hover) { border-color: var(--border-strong); } :host(:focus-within) { outline: var(--border-width-default) solid var(--border-focus); outline-offset: 2px; } </style> <slot></slot> `; } }
