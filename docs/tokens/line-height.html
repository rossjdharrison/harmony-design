Line Height Tokens

Vertical spacing between lines of text for optimal readability

Tight 1.2

Compact line spacing for headings, UI labels, and space-constrained layouts. Creates visual density and emphasizes individual lines.

The quick brown fox jumps over the lazy dog. This text demonstrates tight line height with minimal vertical spacing between lines. Notice how the lines are closer together, making it ideal for headings and short-form content.

Best Used For

  • Headings (h1-h6)
  • Button labels
  • Navigation items
  • Compact UI elements
  • Data tables
/* CSS */ line-height: var(--line-height-tight); // JavaScript import { LINE_HEIGHT } from './tokens/line-height.js'; element.style.lineHeight = LINE_HEIGHT.tight;
Normal 1.5

Standard line spacing for body text and general content. Provides balanced readability without excessive vertical space. This is the default for most text.

The quick brown fox jumps over the lazy dog. This text demonstrates normal line height with standard vertical spacing between lines. This is the recommended default for most body text and provides comfortable reading.

Best Used For

  • Body text (paragraphs)
  • Form fields
  • Card content
  • List items
  • General UI text
/* CSS */ line-height: var(--line-height-normal); // JavaScript import { LINE_HEIGHT } from './tokens/line-height.js'; element.style.lineHeight = LINE_HEIGHT.normal;
Relaxed 1.75

Generous line spacing for long-form content and enhanced readability. Creates breathing room between lines, reducing eye strain during extended reading.

The quick brown fox jumps over the lazy dog. This text demonstrates relaxed line height with increased vertical spacing between lines. Notice the extra breathing room, which makes this ideal for articles and documentation.

Best Used For

  • Long-form articles
  • Documentation
  • Blog posts
  • Accessibility-focused content
  • Reading-intensive interfaces
/* CSS */ line-height: var(--line-height-relaxed); // JavaScript import { LINE_HEIGHT } from './tokens/line-height.js'; element.style.lineHeight = LINE_HEIGHT.relaxed;
