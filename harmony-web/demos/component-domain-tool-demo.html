Component → Domain Links Tool

This tool helps you explore the relationships between UI components and domain types in the Harmony Design System. See what domain entities each component can render and how they're bound together.

About Component → Domain Links

Component → Domain links define how UI components render domain entities from bounded contexts. This tool shows:

  • Primary bindings: The main domain entity a component displays
  • Reference bindings: Related entities shown for context
  • Collection bindings: Lists of entities rendered by the component
  • Props mapping: How component properties map to domain fields
